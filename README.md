# Supernova Wind Simulations

PLUTO hydrodynamics simulations of wind tunnel and stellar wind problems.

## Structure
- `Wind_Tunnel/original/` — baseline Mach 3 wind tunnel
- `Wind_Tunnel/VX1_4.0/` — increased inlet velocity VX1=4.0
- `Wind_Tunnel/VX1_50/` — high velocity VX1=50.0
- `wind_tunnel_script.py` — analysis and comparison animations

## Units (from definitions.h)
- UNIT_DENSITY  = 1.67e-24 g/cm³
- UNIT_VELOCITY = 1.0e5 cm/s (1 km/s)
- UNIT_LENGTH   = 1.496e13 cm (1 AU)
